Formato de texto


Página principal  



Los elementos de formato se diseñaron para mostrar tipos especiales de texto:

<b> - Texto en negrita

<strong> - Texto importante

<i> - Texto en cursiva

<em> - Texto enfatizado

<mark> - Texto marcado

<small> - Texto más pequeño

<del> - Texto eliminado

<ins> - Texto insertado

<sub> - Texto de subíndice

<sup> - Texto en superíndice


Elementos HTML <b> y <strong>

El <b> elemento HTML define texto en negrita, sin ninguna importancia adicional.

Ejemplo: <b>Texto en negritas</b>

Ejemplo: Entre las hojas secas y el viento frío, caminaba ella en busca de un refugio que solo su corazón conocía.

El elemento HTML <strong> define texto con gran importancia. El contenido interior normalmente se muestra en negrita.

Ejemplo: <strong>¡Texto importante!</strong>

Ejemplo: La luna llena iluminaba el cielo estrellado mientras las olas del mar golpeaban suavemente la arena dorada de la playa desierta.

Elementos HTML <i> y <em>

El elemento HTML <i> define una parte del texto en una voz o estado de ánimo alternativo. El contenido interior normalmente se muestra en cursiva.

Sugerencia: la <i> etiqueta se usa a menudo para indicar un término técnico, una frase de otro idioma, un pensamiento, el nombre de un barco, etc.

Ejemplo: <i>Texto en cursiva</i>

Ejemplo: Con sus alas desplegadas y el sol en su rostro, el águila se elevó majestuosamente hacia el cielo azul sin fin, dejando atrás el mundo terrenal y sus preocupaciones.

El elemento HTML <em> define texto enfatizado. El contenido interior normalmente se muestra en cursiva.

Sugerencia: un lector de pantalla pronunciará las palabras con <em> énfasis, usando acento verbal.

Ejemplo: <em>Texto enfatizado</em>

Ejemplo: El viento susurraba suavemente a través de los árboles mientras las hojas de otoño caían en cascada, pintando el suelo de rojo, naranja y dorado en una danza tranquila y armoniosa.

Elemento HTML <small>

El elemento HTML <small> define texto más pequeño:

Ejemplo: <small>Texto pequeño</small>

Ejemplo: Las luces de la ciudad se extendían a lo lejos como un mar de colores mientras los edificios se alzaban hacia el cielo, recordando a los espectadores que en la vida, todo es posible.

Elemento HTML <mark>

El elemento HTML <mark> define el texto que debe marcarse o resaltarse:

Ejemplo: <p>No olvides comprar <mark>leche</mark> antes de llegar</p>

Ejemplo: La nieve cubría el suelo en una manta blanca y suave, transformando el paisaje en un mundo mágico de invierno donde todo era posible, incluso hacer un ángel en la nieve.

Elemento HTML <del>

El elemento HTML <del> define el texto que se ha eliminado de un documento. Los navegadores generalmente marcarán una línea a través del texto eliminado:

Ejemplo: <p>Mi color favorito es el <del>azul</del></p>

Ejemplo: La lluvia caía en gotas gruesas y pesadas, creando un ritmo relajante mientras la naturaleza se regaba y se renovaba, transformando el mundo en un lugar fresco y lleno de vida.

Elemento HTML <ins>

El elemento HTML <ins> define un texto que se ha insertado en un documento. Los navegadores normalmente subrayarán el texto insertado:

Ejemplo: <p>Mi color favorito es el <del>azul</del> <ins>rojo</ins></p>

Ejemplo: El sol se deslizaba suavemente por el horizonte, bañando el cielo en una mezcla de rojos, rosas y dorados, mientras las estrellas comenzaban a despertar y el mundo se sumía en una tranquilidad sublime.

Elemento HTML <sub>

El elemento HTML <sub> define el texto del subíndice. El texto del subíndice aparece medio carácter por debajo de la línea normal y, a veces, se representa en una fuente más pequeña. El texto de subíndice se puede utilizar para fórmulas químicas, como H2O:

Ejemplo: <p>Este es un <sub>subíndice</sub></p>

Ejemplo: El viento soplaba con fuerza, agitando los árboles y las hojas, mientras las nubes oscuras se arremolinaban en el cielo, anunciando la llegada de una tormenta poderosa que transformaría el paisaje en un espectáculo salvaje y majestuoso.

Elemento HTML <sup>

El elemento HTML <sup> define texto en superíndice. El texto en superíndice aparece medio carácter por encima de la línea normal y, a veces, se representa en una fuente más pequeña. El texto en superíndice se puede usar para notas al pie, como WWW[1]:

Ejemplo: <p>Este es un <sup>superíndice</sup></p>

Ejemplo: Los copos de nieve caían lentamente del cielo, cubriendo el mundo en un manto blanco y brillante, mientras el frío mordía la piel y el viento silbaba en los oídos, creando una sensación de paz y quietud que solo se puede encontrar en la naturaleza en invierno.
