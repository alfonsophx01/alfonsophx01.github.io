¿Que es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear y diseñar páginas web. Es el lenguaje base de la web y es interpretado por los navegadores web para mostrar contenido web. HTML utiliza etiquetas (tags) para definir la estructura y el contenido de una página web. Estas etiquetas se utilizan para definir los encabezados, párrafos, enlaces, imágenes, formularios y otros elementos que se utilizan en las páginas web.
